# TarraCoata

Independent TanStack Start + Vite + React + Tailwind project.

## Development

```bash
bun install
bun run dev
bun run build
bun run lint
```

## Environment variables

| Variable | Purpose |
|---|---|
| `RESEND_API_KEY` | Direct Resend email (contact form) |
| `CONTACT_INBOX_EMAIL` | Destination inbox |
| `RESEND_FROM_EMAIL` | Verified sender address |
| `VITE_SUPABASE_URL` | Supabase project URL |
| `VITE_SUPABASE_PUBLISHABLE_KEY` | Supabase anon/publishable key |
